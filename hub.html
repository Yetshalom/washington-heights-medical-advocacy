3) I need help getting to appointments (transportation)

We can help plan routes and connect you to options.

What advocates can do
  • Find subway/bus directions
  • Help apply/schedule Access-A-Ride (paratransit)
  • If eligible, help request Medicaid non-emergency medical transportation (NEMT)
  • Ask the clinic/hospital social worker about transportation programs
Request Help

Note: We do not personally transport patients.

4) My medications are too expensive

We can help you look for cheaper options and assistance.

Ways to lower medication costs
  • Check discount cards (e.g., GoodRx / SingleCare) and compare pharmacies
  • Ask if a generic exists
  • Ask the prescriber about cheaper alternatives
  • Look for manufacturer assistance programs (when available)
  • For seniors: check Medicare Part D plan options and Extra Help eligibility
Request Help

We can help with comparisons, but we don’t sell insurance or take payments.
